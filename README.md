🛠 Especificaciones Técnicas del ADN de Somalia
Este contrato documenta la absorción de nutrientes y la arquitectura aromática:

Índice de Fragancia (uint256): Cuantifica la complejidad del uso de la mezcla Xawaash (comino, cilantro, cardamomo, canela). Este parámetro mide la persistencia aromática del plato.

Capilaridad del Grano (uint256): Registra la capacidad del arroz basmati para absorber caldos sin perder su integridad física (turgencia). Un valor alto indica una hidratación profunda y eficiente.

Usa Carne con Hueso (bool): Validador técnico del uso de cortes con hueso para la extracción de colágeno, lo que aumenta la viscosidad y el valor nutricional del caldo base.

Sistema de Feedback: Funciones de darLike y darDislike integradas para la interacción comunitaria on-chain.
